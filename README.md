[![Build Status](https://travis-ci.org/byrdware/rester.svg?branch=master)](https://travis-ci.org/byrdware/rester)

## Installation

  `npm install -g @byrdware/rester`

## Usage

    $ rester postman [-t 'Test Name'] ./rest/test-suite.json
  
## Tests

  `npm test`
